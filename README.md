# Calculadora Simple

Este proyecto consiste en una calculadora simple que emula las funcionalidades de una calculadora, desarrollada en el marco de la materia Práctica Profesionalizante I. El objetivo fue poner en práctica los conocimientos adquiridos en HTML, CSS y JavaScript a lo largo de dos semanas.

## Descripción

La calculadora permite realizar operaciones básicas (suma, resta, multiplicación y división) y cuenta con un diseño personalizado que mejora la experiencia del usuario. Además, se implementaron características para guardar el número, el operador y el resultado, lo que permite mostrar un historial de operaciones realizadas.

## Desafíos enfrentados

- **Guardar el número, el operador y el resultado:** Implementar una lógica para almacenar estos elementos y mostrarlos en el historial.
- **Ordenar los botones:** Asegurar que la disposición de los botones sea intuitiva y fácil de usar.
- **Diseño personalizado:** Crear un diseño atractivo y funcional que se adapte a diferentes tamaños de pantalla.

## Instalación

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/Mijaliv/calculadora.git
   ```
2. Abrir el archivo `index.html` en un navegador web.
3. Opcional: Solamente ir a https://mijaliv.github.io/calculadora/

## Uso

1. Ingrese los números y seleccione el operador deseado.
2. Presione el botón "=" para obtener el resultado.
3. Consulte el historial de operaciones para ver los cálculos realizados.
